# Task Pick-Rank — Implementation Plan (supersedes cell-rank)

**Goal:** Each task card carries a `0–4` strip (Sketch 005, Variant A: footer strip + corner number). 1–4 = pick order *within the card's intersection cell* (status×priority); 0 = unranked. Numbers unique per cell. Cards sort: ranked (1→4) → unranked by due date (soonest first) → no-due in drag order. Moving a card to another cell resets its rank and bumps the old cell up. Persisted per user. Works on desktop board + mobile drill list (editable); list view shows a read-only badge.

**Why a rewrite:** the earlier build ranked *cells*; correct target is *tasks*, scoped within their cell.

**Storage:** `state.taskRanks = { taskId: { r: 1..4, cell: "status:priority" } }` in Supabase `user_metadata.taskRanks` + localStorage. No schema change; task rows untouched. The stored `cell` lets render detect a move (current cell ≠ stored cell → reset).

### Task 2: Helpers
```js
const cellKey = (s,p) => s+':'+p;
function rankOf(id){ const e=state.taskRanks[id]; return e?e.r:0; }
// Reconcile on every render: drop ranks for deleted/moved tasks, then renumber
// each cell's surviving ranked tasks 1..k by current rank (auto-bump on departure).
function reconcileTaskRanks(){
  let changed=false;
  const byCell={};
  for(const id of Object.keys(state.taskRanks)){
    const t=state.tasks.find(x=>x.id===id);
    const e=state.taskRanks[id];
    if(!t || cellKey(t.status,t.priority)!==e.cell){ delete state.taskRanks[id]; changed=true; continue; }
    (byCell[e.cell]=byCell[e.cell]||[]).push([id,e.r]);
  }
  for(const cell of Object.keys(byCell)){
    byCell[cell].sort((a,b)=>a[1]-b[1]).forEach(([id],i)=>{
      if(state.taskRanks[id].r!==i+1){ state.taskRanks[id].r=i+1; changed=true; }
    });
  }
  return changed;
}
function setTaskRank(id, rank){
  const t=state.tasks.find(x=>x.id===id); if(!t) return;
  const cell=cellKey(t.status,t.priority);
  if(!rank){ delete state.taskRanks[id]; }
  else {
    // evict same-cell holder of this rank
    for(const k of Object.keys(state.taskRanks)){
      const e=state.taskRanks[k]; if(e.cell===cell && e.r===rank && k!==id) delete state.taskRanks[k];
    }
    if(rank>4) rank=4;
    state.taskRanks[id]={ r:rank, cell };
  }
  saveProfileSettings(); renderTasks();
}
```

## Decisions (Q&A)
Per-task strip, scoped to cell (correction). Move → reset + old cell bumps (Q2). 1–4 max, 0=clear (Q3). Sort: ranked→due→drag (Q4). Mobile editable (Q5). Variant A (sketch 005). List view = read-only badge (my call, flagged).
## Risk
Render reconcile renumbers each cell to contiguous 1..k — collapses manual gaps and a lone "4" becomes "1". Matches "clear linear order"; flag if you want gaps preserved.
