# Cell Pick-Rank — Implementation Plan

**Goal:** Each of the 9 actionable intersection cells (Backlog / To Do / In Progress × High/Med/Low) on the desktop board gets an always-visible `0–4` rank strip (Sketch 004, Variant A). 1–4 = pick order (1 = next), 0 = clear. Ranks are unique (assigning evicts the prior holder). When a ranked cell empties, higher ranks shift up by 1 (gap closes). Persisted per user. Mobile tiles show a read-only rank badge. Completed column excluded. Matrix never reorders.

**Data:** `state.cellRanks = { "status:priority": rank }`, e.g. `{ "todo:high": 1 }`. Keys only for status in `['backlog','todo','in-progress']`. Stored in Supabase `user_metadata.cellRanks` + localStorage mirror — no schema change. Task rows untouched.

---

### Task 1: State + persistence
- [ ] `state.cellRanks = {}`.
- [ ] `loadProfileSettings(meta)`: `state.cellRanks = meta.cellRanks || JSON.parse(localStorage.getItem('checkmate_cell_ranks')||'{}')`.
- [ ] `saveProfileSettings()`: add `cellRanks: state.cellRanks` to the `updateUser` payload; mirror `localStorage.setItem('checkmate_cell_ranks', JSON.stringify(state.cellRanks))`.

### Task 2: Helpers
```js
const RANKABLE = ['backlog','todo','in-progress'];
const cellKey = (s,p) => s+':'+p;
function setCellRank(status, priority, rank){
  const key = cellKey(status,priority);
  if(rank===0){ delete state.cellRanks[key]; }
  else {
    for(const k of Object.keys(state.cellRanks)) if(state.cellRanks[k]===rank) delete state.cellRanks[k]; // evict
    state.cellRanks[key]=rank;
  }
  saveProfileSettings(); renderTasks();
}
// On empty: drop ranks whose cell now has 0 tasks; shift higher survivors up by the
// count of removed lower ranks (preserves order + any manual gaps among survivors).
function compactCellRanks(){
  const counts={}; state.tasks.forEach(t=>{const k=cellKey(t.status,t.priority); counts[k]=(counts[k]||0)+1;});
  const removed = Object.entries(state.cellRanks).filter(([k])=>!(counts[k]>0)).map(([,r])=>r);
  if(!removed.length) return false;
  const next={};
  Object.entries(state.cellRanks).filter(([k])=>counts[k]>0).forEach(([k,r])=>{
    next[k] = r - removed.filter(rr=>rr<r).length;
  });
  state.cellRanks = next; return true;
}
```

## Decisions (from Q&A)
Display = badge+emphasis, matrix fixed (Q1). 1=next, 0=clear (Q2). 9 cells, no Completed (Q3). Persist, cloud-synced (Q4). Evict on collision (Q5). Desktop strip + mobile read-only badge (Q6). Empty → shift higher up by 1 (Q7). Variant A (sketch 004).

## Notes / risks
- Manual gaps (e.g. set 1 and 4, skip 2,3) are preserved; only emptied-cell gaps auto-close. If you want strict contiguity always, say so — swap `compactCellRanks` for a full 1..n renumber.
- `compactCellRanks` uses unfiltered task counts, so category/search filters don't disturb ranks.
- Strip adds a thin header to each rankable cell (slightly less card height) — accepted trade-off for always-on retagging.
