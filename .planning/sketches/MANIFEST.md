# Sketch Manifest

## Design Direction
Mobile redesign of Checkmate (static HTML/CSS/JS task manager). Clean-flat, light-default aesthetic reusing the app's existing color tokens. Mobile board defaults to a 3×3 status×priority count-grid; tapping a cell drills into a full-screen list of that intersection. In-app desktop/mobile toggle renders mobile as a centered ~400px column (no bezel). Title-first quick-add for fast task entry.

## Reference Points
- Existing Checkmate desktop UI (style.css tokens, glassmorphic → flattened for mobile)
- ui-ux-pro-max flat-mobile tokens (no shadows/blur, radius 12/999, 48px targets, press-scale)
